## General info
This is complete authApp using express-session and PassportJS.
	
## Technologies
Project is created with:
* NodeJS
* ExpressJS
* MongoDB
* Mongoose
* Config
* Express-Session
* PassportJS
* Express-Ejs-Layouts
* Bcrypt
	
## Setup
To run this project, install it locally using npm:

```
$ cd authApp-nodejs-expressSessions
$ npm install
$ mongod
$ npm run dev
```
